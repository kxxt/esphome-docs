The `mcp9600`   sensor platform allows you to use your MCP9600 and MCP9601
([datasheet](https://cdn-learn.adafruit.com/assets/assets/000/105/040/original/MCP960X-Data-Sheet-20005426%281%29.pdf),
`Adafruit`_) to measure the temperature of a connected thermocouple while also measuring the ambient temperature around the sensor with ESPHome. [I²C](#i2c) is
required to be set up in your configuration for this sensor to work.

{{< note >}}
The [I²C](#i2c) bus must be set to a minimum of `10kHz`   due to the limitations of the MCP9600 and MCP9601.

{{< /note >}}
{{< img src="mcp9600.jpg" alt="Image" caption="MCP9600 Thermocouple Amplifier" width="80.0%" class="align-center" >}}

```yaml
# Example configuration entry
sensor:
  - platform: mcp9600
    hot_junction:
      name: "Thermocouple Temperature"
    cold_junction:
      name: "Ambient Temperature"
    thermocouple_type: K
    address: 0x67
    update_interval: 60s

```
## Configuration variables:

- **hot_junction** (*Optional*): The information for the Hot Junction temperature sensor.

  - All options from [Sensor](#config-sensor).

- **cold_junction** (*Optional*): The information for the Cold Junction temperature sensor.

  - All options from [Sensor](#config-sensor).

- **thermocouple_type** (*Optional*): Set the thermocouple type. Options are:
  K, J, T, N, S, E, B and R type thermocouple. Defaults to `K`  .
- **address** (*Optional*, int): Manually specify the I²C address of
  the sensor. Defaults to `0x67`  .
- **update_interval** (*Optional*, [Time](#config-time)): The interval to check the
  sensor. Defaults to `60s`  .
