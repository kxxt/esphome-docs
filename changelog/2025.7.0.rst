Web Server OTA
--------------

.. warning::

    Read this breaking change carefully, if your only ota method is ``web_server`` and you don't have it explicitly enabled you
    will lose the ability to update devices OTA after updating to 2025.7.0 without adding the ``web_server`` OTA platform.

The :doc:`/components/web_server` has had the OTA functionality moved to a proper OTA platform: :doc:`/components/ota/web_server`.

.. code-block:: yaml

    ota:
      - platform: ...
      - platform: web_server


ESP-IDF 4.x Support
^^^^^^^^^^^^^^^^^^^

This release removes any support for ESP-IDF 4.x versions. Please see
`our developer blog <https://developers.esphome.io/blog/2025/06/18/sunsetting-support-for-idf-4x/>`__ for more details.

For most users, this will not affect you as we set the recommended version as a default. But if you are explicitly setting the
ESP-IDF version in your configuration, you will need to update it to 5.3.2 or later otherwise your project will most likely fail to compile.


ESP32 Arduino 3
---------------

To align with the current ESP-IDF version of 5.3.2, Arduino for the ESP32 has now been updated to 3.1.3.


Sub Devices
-----------

Entities can now be assigned to a sub device! This allows for a single ESP32 device that may have entities that should be represented
in different areas of your home to be represented in Home Assistant as such.
